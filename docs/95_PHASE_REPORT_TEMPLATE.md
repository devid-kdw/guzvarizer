# Phase Report Template

Koristi ovaj format nakon svake faze.

## 1) Phase ID

Primjer: `FE-2` ili `BE-3`

## 2) Scope completed

- kratko sto je napravljeno
- koje datoteke su promijenjene

## 3) Validation

- koji commandi/testovi su pokrenuti
- glavni rezultat (pass/fail)

## 4) Dependencies

- sto je instalirano u fazi
- je li trebalo global/admin odobrenje

## 5) Issues / blockers

- otvoreni problemi
- rizik i impact

## 6) Handoff notes

- sto treba preuzeti drugi agent ili orkestrator
- prijedlog iduce faze
